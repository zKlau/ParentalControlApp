## Setup
1. Clone de repository
```commandline
    git clone https://github.com/zKlau/ParentalControl.git
    cd ParentalControl
```
2. Build the Project
```commandline
    mvn clean install
```

3. Run the application
```commandline
    mvn javafx:run
```

> Make sure you have Java 21+ and Maven installed and properly configured in your environment.
